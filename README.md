# Digital Asset Protection (Basic Full Stack App)

## 🚀 Features
- Create asset
- View assets
- Edit asset
- Delete asset
- Dashboard stats
- Search functionality

## 🛠 Tech Stack
- React + Tailwind CSS
- Node.js + Express

## ▶️ Run Project

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev  

## ⚠️ Note
- No database used
- Data resets when server restarts
